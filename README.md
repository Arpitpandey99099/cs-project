 Console To-Do List Application
A simple, persistent, command-line application built with Python to manage daily tasks.

Features
This application is designed for quick and easy task management directly from your terminal.

Persistence: Tasks are saved to a local file (todo_list.txt) and automatically loaded when the application restarts.

Add Tasks: Easily append new tasks to your list.

View Tasks: Display all current tasks with sequential numbering.

Delete Tasks: Remove completed or unwanted tasks by referencing their number.

Simple Interface: Runs entirely in the console with a clean, text-based menu.



 How to Use
The application will launch and present you with a main menu.

Example Interaction:
View Tasks: Enter 1. (Will initially be empty)

Add a Task: Enter 2.

Prompt: Enter the new task: Buy groceries

Add Another Task: Enter 2.

Prompt: Enter the new task: Finish Python project

View Tasks Again: Enter 1.

 Your To-Do List:
  1. Buy groceries
  2. Finish Python project
Delete a Task: Enter 3.

Prompt: Enter the number of the task to delete: 1

Exit: Enter 4. (This saves your current list.)
